# Web App

React + Vite + TypeScript + Three.js

## Setup

```bash
npm install
npm run dev
```

## Structure

```
src/
├── components/    # UI components
├── pages/         # Route pages
├── hooks/         # Custom hooks
├── services/      # API calls
├── store/         # Zustand state
├── types/         # TypeScript types
└── utils/         # Helpers
```
